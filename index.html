Política de Privacidad — OK-SHIP

Última actualización: 30 de marzo de 2026

OK-SHIP Consultoría Marítima desarrolló la aplicación móvil OK-SHIP como una herramienta para que sus clientes puedan acceder a información sobre sus embarcaciones, certificados, carenas y observaciones.

Información que recopilamos

La aplicación recopila la siguiente información para su funcionamiento:

  • RUT del usuario (utilizado como identificador de inicio de sesión)
  • Contraseña (almacenada de forma segura y encriptada)
  • Información de embarcaciones asociadas al usuario
  • Certificados, carenas y observaciones de cada embarcación

Uso de la información

La información recopilada se utiliza exclusivamente para:

  • Autenticar al usuario dentro de la aplicación
  • Mostrar la información de embarcaciones y documentación asociada
  • Garantizar que cada usuario acceda únicamente a sus propios datos

Almacenamiento y seguridad

Toda la información es almacenada de forma segura en Supabase, una plataforma de base de datos con cifrado de datos en reposo y en tránsito. Implementamos políticas de seguridad a nivel de fila (Row Level Security) para garantizar que cada usuario acceda únicamente a sus propios datos.

Compartición de datos

OK-SHIP no vende, intercambia ni transfiere información personal a terceros. Los datos son accesibles únicamente por el equipo de OK-SHIP y el usuario propietario de los datos.

Retención de datos

Los datos se conservan mientras el usuario mantenga una cuenta activa en la aplicación. Si desea eliminar su cuenta y datos asociados, puede solicitarlo contactando a OK-SHIP directamente.
